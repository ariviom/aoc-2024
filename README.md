# advent

## Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
```

## Setup

To install dependencies:

```bash
bun install
```

To run each day:

```bash
bun run dir/index.ts
```

This project was created using `bun init` in bun v1.1.38. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
